1. 简介与适用范围

LavaChat 是一款运行于 iOS 16 及以上版本的本地应用。我们致力于最小化数据收集与传输，不运营自有服务器，不进行行为跟踪或画像。本政策说明 LavaChat 在您使用过程中如何处理与保护数据。

2. 我们收集/不收集什么

  • 不收集：第三方统计/分析/崩溃数据（如 Firebase、Sentry、TelemetryDeck）、广告标识符（IDFA/ATT）。
  • 订阅与支付：完全由 Apple 的 StoreKit 处理，LavaChat 不接触您的支付卡信息。
  • CloudKit 使用范围：仅在您主动发起 iCloud 分享时使用，用于生成分享链接；不进行自动云同步。
  • LLM 调用：仅支持用户自带 API Key的直连第三方模型服务商；聊天内容与附件在您指令下发送给该服务商处理。
  • 日志：发布版本不打印或记录网络请求体/聊天内容；如有错误日志，为最小化信息且仅保存在本地，不回传任何服务器。

3. 我们处理的数据及来源

  • 您提供的数据：聊天文本、您选择上传/粘贴的图片或文件、模型与会话配置。
  • 设备权限：
    • 相机（扫码/拍照）：仅在您授权并使用相关功能时访问。
    • 相册（选图）：仅在您授权并选择图片时访问。
  • 订阅相关：订阅状态与交易由 Apple 通过 StoreKit 提供；我们不持有可识别您的支付信息。

4. 处理方式与存储位置

  • 本地存储：应用数据保存在您的设备（例如 Core Data、本地文档/临时目录），偏好设置在 UserDefaults，API Key 保存在 Keychain。
  • 第三方 LLM 服务商：当您发起对话或上传文件时，相关内容会通过加密连接（HTTPS）发送至您配置的模型服务商。不同服务商的数据处理与存储位置受其各自的隐私政策与条款约束。
  • iCloud 分享（CloudKit）：仅当您主动分享时，我们会将导出的配置/会话等 ShareableData 序列化后写入您的 iCloud（作为记录字段）并生成分享链接。您可以接受/停止分享。

5. 第三方服务与跨境传输

您可自行配置并选择第三方模型服务商用于处理请求。出于模型推理与存储的需要，您的数据可能被传输至境外服务器。请在使用前审阅并同意所选服务商的条款与隐私政策，并谨慎避免上传敏感个人信息。

6. 您的权利与控制

  • 在应用内查看、导出、删除本地数据；撤回或停止 iCloud 分享。
  • 随时更换或删除 API Key。
  • 随时撤回相机/相册等系统权限（可能影响相关功能使用）。

7. 数据保留与删除

我们不运营自有服务器，不保存您的对话或文件副本。应用数据保留在您的设备，直到您主动删除或卸载应用。对于 iCloud 分享，您可随时停止分享或删除相关记录。

8. 安全措施

  • API Key 存储在系统 Keychain。
  • 应用运行在 iOS 沙箱环境，网络通信使用 HTTPS。
  • LLM 流式传输优先使用 URLSession 的 .ephemeral 配置以减少持久化缓存。
  • 发布版本不记录请求体/对话内容。

9. 未成年人

本应用不面向 13 岁以下的儿童。如您认为我们在不知情的情况下处理了 13 岁以下儿童的数据，请联系我们以便删除本地数据并指导停止使用。

10. 本政策的变更

我们可能会根据功能或合规要求更新本政策。重大变更将通过应用内或页面更新的方式通知并在本页标注生效日期。若未来启用远程推送，我们将补充说明用途与数据处理方式。
