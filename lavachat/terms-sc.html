1. 与 Apple EULA 的关系

本《使用条款》（“本条款”）为 Apple 标准最终用户许可协议（EULA） 的补充。Apple EULA 规定了您使用本应用的许可与区域性法律条款。如与本条款存在冲突，以 Apple EULA 为准。

2. 服务说明

LavaChat 为一款本地 iOS 应用（iOS 16+），我们不运营自有服务器。应用允许您配置并与大语言模型（LLM）交互；仅在您主动发起请求时才会传输内容。

3. 第三方 LLM 与 API Key

  • 由您自行选择第三方 LLM 服务商并遵守其条款与政策。
  • API Key 由您提供并保存在设备的 iOS Keychain 中；您需对其合法性、保密性、使用及相关费用/配额负责。
  • 当您发送提示词或文件时，将经加密连接（HTTPS）直接传输至您配置的服务商。请避免提交敏感个人信息。
  • 我们不在自有服务器托管、保留或审核您的提示与输出。

4. 订阅与计费（StoreKit）

  • 订阅与支付由 Apple 通过 StoreKit 处理；我们不接触或存储您的支付卡信息。
  • 自动续订与取消在您的 Apple ID 设置内管理；退款（如适用）遵循 Apple 的政策。
  • 除非应用内另行说明，默认不提供试用期。

5. 可接受使用政策（AUP）

您不得利用本应用上传、生成或传输违法、侵权、骚扰、仇恨、歧视、色情、暴力、欺诈、恶意（包括恶意代码或试图绕过安全措施）等内容，或违反任何第三方政策、出口/制裁法律。若我们有合理理由认为存在滥用或违规，可限制或暂停功能访问。

6. AI 输出免责声明

LLM 输出可能不准确或不完整，仅用于信息参考，并不构成任何专业意见（包括法律/医疗/投资等）。您对输出的使用自行承担责任。

7. iCloud 分享

当您明确选择通过 iCloud 分享时，分享数据会序列化至您的 iCloud 用于生成分享链接。我们不托管或审核此类内容，也无法代为删除；分享内容的责任由您自行承担。

8. 责任限制

在适用法律允许的最大范围内，我们不对任何间接、偶然、特殊、后果性、示范性或惩罚性损害承担责任。与本应用相关的我们的累计赔偿责任，不得超过产生该等索赔前连续十二（12）个月内您就本应用订阅实际支付的费用总额（如有）。

9. 年龄限制

本应用不面向 13 岁以下儿童。未成年人应在法定监护的前提下并在法律允许的范围内使用本应用。
